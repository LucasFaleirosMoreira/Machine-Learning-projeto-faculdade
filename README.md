# Machine Learning aplicado para trabalho da faculdade na matéria de IA

Nesse código, estou aplicando machine learning em um dataset real sobre baleias jubarte da Austrália, conseguindo fazer uma análise precisa sobre quais os próximos movimentos dessas baleias, conseguindo prevenir acidentes de embarcações passsando por cima de baleias, monitorando comportamentos estranhos e podendo entender se a baleia está doente ou com algum problema geral.
